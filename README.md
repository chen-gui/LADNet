**LADNet**
======

## Description

This repository provides the implementation of LADNet, a patch-based self-supervised deep learning framework for denoising seismic common reflection point (CRP) gathers without requiring clean reference data. LADNet is designed to suppress random and incoherent noise in pre-stack seismic CRP gathers while preserving coherent and weak reflection events.

Workflow of the proposed patch-based self-supervised learning framework for denoising CRP gather. (A) Training data preparation: the original noisy CRP gather is divided into overlapping 2D patches, and each patch is vectorized and used as both the network input and the self-supervised target. (B) LADNet denoiser: the vectorized patch is processed by stacked feature-learning blocks consisting of fully connected layers, learnable activation functions, soft attention modules, and residual connections. (C) Training and inference: the LADNet is optimized using the SPLC loss; during inference, the denoised vectors are reshaped into 2D patches and merged by overlap-add reconstruction with mean aggregation in overlapping regions.

## Scientific Application

LADNet is designed for scientific and industrial workflows involving pre-stack seismic CRP gathers, where useful reflection events are often contaminated by random noise and incoherent interference. The proposed self-supervised framework enables researchers to denoise seismic gathers directly from noisy observations without the need for curated clean reference data, making it suitable for practical field-data processing.

By enhancing the signal-to-noise ratio and preserving laterally continuous reflection events, LADNet facilitates multiple downstream seismic processing and interpretation tasks, including:

* Pre-stack seismic data conditioning
* CRP gather quality enhancement
* Random-noise attenuation before stacking
* Weak reflection-event preservation
* Seismic imaging and inversion preprocessing
* Field seismic data quality control

## Examples

    The "code" directory currently contains the runnable synthetic CRP gather denoising script.

# Synthetic Example

The synthetic angle-domain gathers are generated by amplitude-versus-offset (AVO) forward modeling based on the Marmousi-2 model. Random noise is added to the clean gathers to generate noisy inputs with an SNR of 2 dB. LADNet is compared with FXDecon, DIP, and Self2Self.

Run the synthetic example using:

    python code/syn_gather_pro.py

Generated by [code/syn_gather_pro.py](https://github.com/chen-gui/LADNet/blob/main/code/syn_gather_pro.py)

## Reproducibility

The main parameters used in the synthetic experiment are:

    Patch size: 8 × 8

    Shift size: 1 × 1

    Batch size: 512

    Learning rate: 5e-4

    Optimizer: Adam

    Training epochs: 10

    Loss function: SPLC loss

The default parameters used for field CRP gathers in the paper are:

    Patch size: 20 × 20

    Shift size: 2 × 1

    Batch size: 512

    Learning rate: 5e-4

    Optimizer: Adam

    Training epochs: 30

    Loss function: SPLC loss

## Data

The synthetic data and scripts used in the demo are stored in the "data" and "demo" directories.

The field CRP gather data used in the paper are not included in this repository due to data-size and data-sharing restrictions. This repository focuses on the synthetic example, model implementation, and reproducible demonstration scripts.
